Tetris

Game Instructions:

Tetris with Nintendo rotation system, aka right handed rotation system. Link: https://strategywiki.org/wiki/Tetris/Rotation_systems

Enter P when Power Up is available to trigger your special block-deleting buff. For 5 seconds you can delete any block you like by clicking on it ,but you don't get any points. This buff has a 10-second cooldown.

Garbage rows regularly spawn at the bottom. Beware, garbage rows spawn faster as you level up!

Adapted from Nintendo Scoring system:
Tetromino     = 40
Single Line   = 40*level
Double Line  = 100*level
Triple line      = 300*level
Tetris             = 1200*level
